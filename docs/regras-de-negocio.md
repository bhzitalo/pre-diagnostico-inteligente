# 🧠 Regras de Negócio — PrInt

## 📌 Regras Gerais

- o sistema não substitui um profissional de saúde  
- todas as respostas devem conter aviso de orientação  
- o sistema deve priorizar segurança do usuário  

## 🚦 Classificação de Urgência

### 🟢 Baixa
- sintomas leves  
- recomendação: acompanhar  

### 🟡 Moderada
- sintomas persistentes ou moderados  
- recomendação: procurar atendimento  

### 🔴 Alta
- sintomas graves  
- recomendação: atendimento imediato  

## 🧾 Registro de Dados

- todo atendimento deve ser armazenado  
- os dados devem ser associados ao usuário  
- o usuário pode acessar seu histórico  

## 🔒 Privacidade

- dados devem ser protegidos  
- acesso restrito ao próprio usuário  
- conformidade com LGPD  

## ⚠️ Limitações

- não fornecer diagnósticos  
- não substituir avaliação médica  
- não garantir precisão clínica absoluta
