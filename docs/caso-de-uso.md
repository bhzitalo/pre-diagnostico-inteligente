# 🎭 Casos de Uso — PrInt

## 👤 Ator Principal
Usuário

## 📌 Caso de Uso 1: Cadastro

**Descrição:**  
Usuário cria uma conta no sistema.

**Fluxo:**
1. usuário acessa tela de cadastro  
2. preenche dados  
3. sistema valida  
4. conta é criada  

## 📌 Caso de Uso 2: Login

**Descrição:**  
Usuário acessa o sistema.

**Fluxo:**
1. usuário informa email e senha  
2. sistema valida  
3. acesso liberado  

## 📌 Caso de Uso 3: Iniciar Atendimento

**Descrição:**  
Usuário inicia análise de sintomas.

**Fluxo:**
1. usuário abre chat  
2. informa sintomas  
3. sistema faz perguntas  
4. sistema analisa respostas  
5. sistema retorna resultado  

## 📌 Caso de Uso 4: Visualizar Resultado

**Descrição:**  
Usuário recebe orientação.

**Saída:**
- nível de urgência  
- recomendação  

## 📌 Caso de Uso 5: Consultar Histórico

**Descrição:**  
Usuário visualiza atendimentos anteriores.

**Fluxo:**
1. usuário acessa histórico  
2. sistema exibe registros  

## 📌 Caso de Uso 6: Logout

**Descrição:**  
Usuário encerra sessão.

**Fluxo:**
1. usuário clica em sair  
2. sistema encerra sessão
